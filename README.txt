To run the ANN examples, enter the top level directory (should be multiple .py files and folders src, examples, and tests) and type either of the following commands:

python3 iris.py
python3 fraud.py

To run the Linear Classifier examples, enter the top level directory and type any of the following commands:

python3 earthquake_clean.py perceptron
python3 earthquake_clean.py logistic
python3 earthquake_noisy.py perceptron
python3 earthquake_noisy.py logistic
